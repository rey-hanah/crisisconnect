# 🆘 Crisis Connect

> Peer-to-peer crisis coordination. Map-first. AI-powered. Voice-enabled. No middleman.

**Live at:** crisisconnect.tech  
**Stack:** React + Vite · TypeScript · Tailwind CSS · ShadCN UI · NestJS · MongoDB Atlas · Claude API · ElevenLabs

> "Drop a pin when you need help. Drop a pin when you can help. AI sorts who needs it most."
